# SILOG
SILOG (Sistem Informasi Logistik) adalah aplikasi yang dibangun dengan Python untuk memodelkan proses logistik. Aplikasi ini mengelola jaringan kota hub, rute antar kota, serta administrasi resi pengiriman.Program ini dirancang untuk tugas akhir mata kuliah Struktur Data, dengan aturan khusus sesuai NIM genap.
